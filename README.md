# 📊 Data Analysis Project

A Python-based data analysis project that demonstrates data preprocessing, statistical analysis, and data visualization techniques using real-world analytical workflows. The project analyzes student performance data and presents insights through multiple visualizations.

---

## 🚀 Project Objectives

* Perform data analysis on student performance data.
* Generate meaningful insights from raw datasets.
* Visualize data using different charting techniques.
* Demonstrate practical usage of Python for data analytics.

---

## 🛠️ Technologies Used

| Technology | Purpose                      |
| ---------- | ---------------------------- |
| Python     | Core Programming Language    |
| Pandas     | Data Manipulation & Analysis |
| NumPy      | Numerical Computing          |
| Matplotlib | Data Visualization           |
| CSV        | Dataset Storage              |
